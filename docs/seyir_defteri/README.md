# İmleç Site Seyir Defteri

Bu klasör `imlec-site` platformunun yaşayan teknik hafızasıdır.

FIS260 masaüstü uygulamasının ödeme, üyelik, cihaz, güncelleme, destek bildirimi ve admin panel ihtiyaçları bu platform üzerinden yönetilir. Bu yüzden her sürümde site tarafındaki kararlar ayrıca yazılır.

## Kural

Her sürümde yeni dosya açılır. Eski dosya değiştirilmez.

Dosya adı biçimi:

```text
imlec-site_v<surum>_<YYYYMMDD>.md
```
